test.py 是根据关键帧密度剪切视频，
pyscene.py 是提取视频中所有的场景图片，
places365 是进行场景分类的文件夹，
classify.py 是根据识别的结果将图片移动到对应的文件夹中



短视频生成服务
短视频生成要求对视频素材资源进行智能分析，
自动识别具有较高新闻价值的事件，
自动生成特定片段的视频集锦，
然后从各类视频中提取最关注部分进行自动剪辑合成。
短视频生成服务具体技术指标要求如下：
1. 能够自动从视频流中识别对应的内容并剪切；
2. 支持在线视频流、视频文件（avi、MP4、MPEG、WMV等）、远程监控设备的直接视频读取；
3. 能够对视频内容和场景进行识别，支持的自定义场景大于400个；
4. 支持视频归档、搜索、展示和交互

python pyscene
python extract_name.py
python places365/vgg16_places_365.py
python classify.py

下载视频到当前目录。
